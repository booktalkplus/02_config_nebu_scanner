# 네부스캐너 통합 원격 설정 저장소 (v1.0.1)

이 저장소는 **네부스캐너(Nebu Scanner) 크롬 확장 프로그램**의 실시간 공지사항, 시스템 설정, 메시지 템플릿을 중앙에서 관리하기 위한 전용 저장소입니다.

## 🚀 역할 및 목적
- **심사 없는 실시간 업데이트**: 크롬 웹스토어의 긴 심사 과정을 거치지 않고, 공지사항이나 링크 주소 등을 즉시 수정하여 사용자에게 배포할 수 있습니다.
- **다이나믹 UI 제어**: 오픈톡 참여 코드 유무에 따른 리포트 디자인 변경 등 앱의 동작을 원격에서 제어합니다.

## 📁 파일 구조 및 설명

### 1. `notice.json` (긴급 공지사항)
앱 실행 시 팝업 상단에 표시되는 긴급 공지사항을 제어합니다.
- `show`: `true`로 설정 시 공지 바가 나타납니다.
- `text`: 공지 내용 (HTML 태그 지원).
- `color`: 공지 바의 배경색.

### 2. `config.json` (시스템 설정)
앱에서 사용하는 각종 링크와 정렬 가중치 등을 관리합니다.
- `SUPPORT_KAKAOTALK_CODE`: 오픈톡 참여 코드. ("없음" 설정 시 디자인 자동 전환)
- `EXTERNAL_LINKS`: 쿠팡 파트너스 등 각종 외부 후원 링크.

### 3. `messages.json` (로딩 및 알림 메시지)
분석 중 표시되는 로딩 메시지나 사용자 알림 텍스트를 관리합니다.
- `loading_messages`: 분석 진행 중 무작위로 추출되어 표시될 메시지 목록.

## ⚠️ 주의사항
1. **JSON 문법 준수**: 수정 시 반드시 `json` 문법이 유효하도록 쉼표(,)나 따옴표(") 확인이 필요합니다.
2. **Raw 링크 유지**: 확장 프로그램은 각 파일의 `GitHub Raw` URL을 통해 데이터를 읽어옵니다. 파일명 변경 시 확장 프로그램 소스 코드의 `config.js` 수정이 동반되어야 합니다.
3. **캐시 지연**: 깃허브 서버 캐시 정책에 따라 수정 후 앱에 반영되기까지 약 1~5분 정도의 지연이 발생할 수 있습니다.
