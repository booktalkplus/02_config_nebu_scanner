# 네부스캐너 개인정보 처리방침

**최종 수정일**: 2026년 2월 9일

## 1. 개인정보의 수집 및 이용 목적

네부스캐너("본 서비스")는 다음의 목적을 위해 최소한의 정보를 수집합니다:

- 서비스 이용 통계 분석 (Google Analytics 4)
- 서비스 개선 및 사용자 경험 향상
- 오류 추적 및 기술 지원

## 2. 수집하는 정보

### 자동 수집 정보

- **익명 사용자 ID**: 개별 사용자를 식별하지 않는 무작위 생성 ID
- **사용 통계**: 페이지 조회, 기능 사용 빈도
- **기술 정보**: 브라우저 버전, 운영체제 (개인 식별 불가)

### 수집하지 않는 정보

- 이름, 이메일, 전화번호 등 개인 식별 정보
- 검색한 매물의 상세 주소
- 사용자의 위치 정보
- 쿠키 (확장 프로그램 특성상 불필요)

## 3. 정보의 보관 및 파기

- Google Analytics에 전송된 데이터는 Google의 개인정보 처리방침에 따라 관리됩니다
- 사용자가 확장 프로그램을 제거하면 로컬 저장 데이터는 즉시 삭제됩니다

## 4. 제3자 제공

수집된 정보는 다음을 제외하고 제3자에게 제공되지 않습니다:

- **Google Analytics**: 서비스 분석 목적 (익명화된 통계만 전송)
- **쿠팡 파트너스**: 후원 링크 클릭 시 (사용자 동의 하에 이동)

## 5. 사용자의 권리

사용자는 언제든지:

- 확장 프로그램을 제거하여 데이터 수집을 중단할 수 있습니다
- Chrome 설정에서 확장 프로그램 권한을 관리할 수 있습니다
